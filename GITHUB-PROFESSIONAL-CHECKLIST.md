# GitHub professional checklist

## Repository

- [ ] Repository name is `spirit-terminal-portfolio`
- [ ] Repository is public when ready to show employers
- [ ] Description is complete
- [ ] Twelve relevant topics are added
- [ ] Social preview image is uploaded
- [ ] README renders correctly
- [ ] Actions build has a green check
- [ ] No `node_modules`, `dist`, `.env`, or SQLite database files are committed
- [ ] `v0.4.0` release is published

## Profile

- [ ] Real display name is present
- [ ] Short professional bio is present
- [ ] Portfolio repository is pinned
- [ ] CardSlate release repository is pinned when appropriate
- [ ] Profile photo or professional avatar is present
- [ ] Portfolio URL is added after deployment

## Before sharing

- [ ] `npm ci` succeeds
- [ ] `npm run build` succeeds
- [ ] Desktop and mobile layouts are checked in a real browser
- [ ] GitHub and Discord links work
- [ ] Contact form behavior is checked on the deployed host
- [ ] README screenshots and descriptions match the current build
